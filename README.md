## 📝 Sobre o Projeto

O projeto consiste em um pequeno CRUD (Create, Read, Update, Delete) de Cadastro de Alunos utilizando **Vue 3** com a Composition API (`<script setup>`).

## 🛠️ Como Executar

O código fonte está dentro da pasta `cadastro-alunos`. Para rodar o projeto, siga os passos:

1. Acesse a pasta do projeto:
```bash
cd cadastro-alunos
```

2. Instale as dependências:
```bash
npm install
```

3. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

4. Abra o navegador no endereço indicado (geralmente `http://localhost:5173/`).
