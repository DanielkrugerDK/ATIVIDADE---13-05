# Atividade Prática - CRUD de Alunos (Vue 3)

Esta é a atividade da disciplina **Frameworks Modernos para Desenvolvimento de Sistemas**, desenvolvida por **Gabriel Guedes Rodrigues** e **Daniel Kruger Ferreira Cardoso**.

## 📝 Sobre o Projeto

O projeto consiste em um pequeno CRUD (Create, Read, Update, Delete) de Cadastro de Alunos utilizando **Vue 3** com a Composition API (`<script setup>`).

A atividade foi desenhada para aplicar as principais diretivas do Vue vistas em sala de aula:
- `ref()` para reatividade
- Interpolação `{{ }}`
- `v-if`, `v-else` e `v-show` para renderização condicional
- `v-for` com `:key` para listagens
- `v-model` para two-way data binding no formulário
- `@click` para tratamento de eventos

## 🛠️ Como Executar

1. Instale as dependências:
```bash
npm install
```

2. Rode o servidor de desenvolvimento:
```bash
npm run dev
```

3. Abra o navegador no endereço indicado (geralmente `http://localhost:5173/`).

## ⚙️ Tecnologias Utilizadas

- Vue 3
- Vite
- HTML/CSS nativos (sem frameworks adicionais de UI)
